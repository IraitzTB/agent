# Product Overview

An educational AI agent that helps students learn how to code AI agents for various purposes.

## Purpose

This agent responds to student questions about AI agent development, providing guidance, examples, and explanations on building agents for different use cases.

## Key Characteristics

- Educational focus: Teaching AI agent development concepts
- Interactive Q&A capabilities for student learning
- Kiro-native development environment
- Spec-driven development workflow support
- MCP (Model Context Protocol) integration capability (currently disabled)
